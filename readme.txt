Level editor !!

Wasd to control the character

Q to save your file you created
E to load a file you saved

Left control to backtrack a block you made

Z to create a beginning in a level
X to create an end in a level

Left click to make an obstacle
right click to move your character by hand (or should i say by mouse (im gonna fucking kill myself))
